# 지뢰찾기

GitHub Pages에 바로 배포할 수 있는 정적 지뢰찾기 게임입니다.

## 실행

`index.html`을 브라우저에서 열면 됩니다. 별도 빌드나 패키지 설치가 필요 없습니다.

## 배포

1. 이 폴더를 GitHub 저장소로 푸시합니다.
2. GitHub 저장소의 `Settings` > `Pages`로 이동합니다.
3. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
4. 배포 브랜치를 `main`, 폴더를 `/root`로 선택하고 저장합니다.

## 파일

- `index.html`: 화면 구조
- `styles.css`: 반응형 UI 스타일
- `script.js`: 지뢰찾기 게임 로직
